An advanced stopwatch for Wordpress

== Description ==

Adv Stopwatch let's you add an advanced online stopwatch to your site. 

Just like the one you can see here: <a href="http://www.findhold.dk/stopur/">http://www.findhold.dk/stopur/</a>

The plugin was developed for <a href="http://www.findhold.dk/">Find Hold</a> by <a href="http://bryancarraghan.com/">Bryan Carraghan</a>.

== Installation ==

1. Install and activate UV Stopwatch like any other plugin (i.e. first use the Wordpress plugin installer or FTP to install the plugin. Then click the Activate link to activate the plugin)
2. Add the shortcode [stopwatch] to the page/post where you want to display the stopwatch.

Additional info: <a href="http://www.findhold.dk/adv-stopwatch-a-wordpress-stopwatch-plugin/">http://www.findhold.dk/adv-stopwatch-a-wordpress-stopwatch-plugin/</a>
